K Editor :: Config :: TableInsertDragSize

TableInsertDragSize

표 삽입 메뉴의 최대 선택 개수를 설정합니다.

remarks

기본값은 "10,10" 이고, "행,열" 개수로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div border="0" style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 표 삽입 메뉴의 최대 선택 개수를 15행,10열로 설정합니다.
            TableInsertDragSize: '15,10'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
