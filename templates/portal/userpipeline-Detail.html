Bio-express's Single-cell-RNA-Sequencing-Pipeline is an extensible toolkit for analyzing single-cell gene expression data using the Scanpy pipeline.
It includes methods for preprocessing, visualization, clustering, differential expression testing.
Its Python-based implementation efficiently deals with data sets of more than one million cells.
we present ANNDATA, a generic class for handling annotated data matrices.

This pipeline features
1) We regress out confounding variables, normalize, and identify highly variable genes.
2) TSNE and graph-drawing (Fruchterman–Reingold) visualizations show cell-type annotations obtained by comparisons with bulk expression.
3) Cells are clustered and plotted using the Louvain algorithm. It also supports clustering using various algorithms.
4) Ranking the differentially expressed genes in clusters identifies marker genes that match the bulk label
